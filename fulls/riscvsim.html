A collection of makefiles and .c files that simplify the workflow of RISC-V emulation and encapsulate the QEMU RISC-V simulator and GNU toolchain.
This project gives me a chance to delve deeper into how an emulator simulates a CPU and the details of the workflow for cross-compiling for a RISC-V CPU on an x86 host.

Relevant Skills

QEMU GNU-toolchain RISC-V GNU make C/C++ C standard library
